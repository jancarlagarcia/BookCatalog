# BookCatalog

BookCatalog é um projeto em Python desenvolvido para catalogar e gerenciar um acervo pessoal de livros diretamente pelo terminal.

O sistema permite cadastrar livros, organizar leituras, marcar livros como finalizados, avaliar com estrelas e manter os dados salvos entre execuções por meio de persistência em arquivo JSON.

Este projeto foi criado com foco em aprendizado prático de Python, lógica de programação, organização de código e persistência de dados.

---

## Funcionalidades

- Cadastrar livros com:
  - Título
  - Autor
  - Gênero
- Listar livros não lidos, organizados por gênero
- Marcar livros como finalizados
- Avaliar livros finalizados (1 a 5 estrelas)
- Listar livros finalizados ordenados por avaliação
- Listar todo o acervo em ordem alfabética
- Excluir livros do acervo com confirmação
- Persistência de dados em arquivo JSON (`acervo.json`)
